# Interactive Project: Plankton Microcosm
You can try demo here: https://imgeyou.github.io/Interactive_Plankton/final/index.html

# Interactions:
1. Default: wave your hands, the planktons wil shy away from human hands
2. Point gesture (extending only index finger): new gold particels will generate from the index fingertip and it will lead the movement of some planktons.
3. Fox Gesture (https://en.wikipedia.org/wiki/Silent_fox_gesture) + Sound trigger (sudden increase in volume): the planktons will be blown away out of the screen. But they will resurface after a few seconds.

## notes: 
When posing fox gesture, try not to let index finger and pinky finger overlap. This will ensure better gesture recognition.
